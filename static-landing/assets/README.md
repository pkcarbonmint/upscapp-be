# Image Assets for Landing Page

This directory contains image assets for the Study Planner landing page.

## Required Images:

1. **favicon.ico** - Website favicon (16x16, 32x32, 48x48 px)
2. **og-image.png** - Social media preview image (1200x630 px)
3. **hero-bg.svg** - Optional: Hero section background illustration
4. **feature-icons/** - Optional: Custom feature icons

## Notes:
- The current landing page uses inline SVG icons, so no external image files are strictly required
- You can add custom illustrations or photos to enhance the visual appeal
- Ensure all images are optimized for web (compressed, appropriate format)
- Consider WebP format for better compression

## Image Optimization Tips:
- Use WebP format when possible for better compression
- Include alt text for accessibility
- Use appropriate sizes for different screen densities
- Consider lazy loading for below-the-fold images
